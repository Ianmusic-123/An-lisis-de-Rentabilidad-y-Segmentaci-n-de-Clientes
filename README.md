# An-lisis-de-Rentabilidad-y-Segmentaci-n-de-Clientes-E-commerce-
Este proyecto simula un entorno real de análisis de datos para una plataforma de E-commerce. El objetivo principal es transformar datos transaccionales brutos en estrategias de retención y crecimiento mediante el uso de Python para el procesamiento y SQL para la segmentación avanzada de clientes.
